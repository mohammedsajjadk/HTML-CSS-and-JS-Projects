# Installation and how to run this project

1. Go to any folder --> Open Command Prompt --> Clone the project from github. Use
   ```
   git clone https://github.com/mohammedsajjadk/HTML-CSS-and-JS-Projects.git
   ```
2. Either go inside that folder or copy the contents inside that folder to the folder you want.
3. Enter this command
   ```
   git checkout Budgety
   ```
4. Run the below command
   ```
   npm install
   ```
5. Open command prompt (in the project root directory) -> Type **_npm run start_** and hit Enter
6. In the browser, localhost:8080 or some other localhost port will open
